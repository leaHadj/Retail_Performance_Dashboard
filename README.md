# Retail Performance Dashboard – Power BI

## Project Overview
This project analyzes retail sales performance using the Sample Superstore dataset.

The objective is to build a clear and interactive dashboard to visualize key business metrics.

---

## Tools Used
- Excel (data cleaning)
- Power Query
- Power BI
- DAX (measures creation)

---

## Key KPIs
- Total Sales
- Total Profit
- Profit Margin %
- Sales by Region
- Sales by Category
- Sales by Month
- Top Performing Products

---

## Project Structure

01_Data_Raw → Original dataset  
02_Data_Cleaned → Cleaned Excel file  
03_PowerBI → Power BI dashboard (.pbix)  
04_Exports → Dashboard screenshots  

---

## Insights
- West region generates the highest sales
- Technology is the most profitable category
- Sales show monthly fluctuations
- A small group of products drives a large share of revenue
